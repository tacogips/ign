## Overview

This is a template generated by `ign template new`.

## Variables

This template uses the following variables:

- `project_name` (string, required): Name of the project
- `description` (string, optional): Project description
- `enable_feature` (bool, optional): Enable optional feature

## Usage

1. Edit `ign.json` to define your template variables
2. Add your template files with ign directives
3. Test with `ign template check`

## Ign Directives

### Variable Substitution
```
@ign-var:variable_name@
@ign-var:variable_name:string@
@ign-var:variable_name=default_value@
@ign-var:variable_name:int=8080@
```

### Conditionals
```
@ign-if:enable_feature@
This content appears when enable_feature is true.
@ign-endif@

@ign-if:enable_feature@
Feature enabled content.
@ign-else@
Feature disabled content.
@ign-endif@
```

### Comments (removed from output)
```
@ign-comment: This is a template comment @
```

### Raw Content (preserved literally)
```
@ign-raw: @ign-var:not_processed@ @
```

### File Inclusion
```
@ign-include:path/to/file.txt@
```
